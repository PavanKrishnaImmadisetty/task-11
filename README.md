## Overview
This project uses **Support Vector Machine (SVM)** to classify breast cancer data as **benign or malignant** using the Sklearn Breast Cancer dataset.

---

## Dataset
- Source: `sklearn.datasets.load_breast_cancer()`
- Features: 30
- Classes: Malignant (0), Benign (1)

---

## Steps
- Data loading and exploration
- Feature scaling using StandardScaler
- SVM with linear and RBF kernels
- Hyperparameter tuning using GridSearchCV
- Evaluation using confusion matrix, ROC curve, and AUC

---

- Python
- Scikit-learn
